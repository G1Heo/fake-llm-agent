# 🤖 Fake LLM Agent 프로젝트

이 프로젝트는 파이썬을 활용해 간단한 LLM 에이전트의 동작 원리를 학습하기 위한 저장소입니다.

## 🎯 프로젝트 목적
* Git 브랜치 전략 및 협업 흐름 익히기
* 파이썬 클래스와 모듈 구조 이해

## 📚 학습 내용
1. **Git Revert**: 잘못된 커밋을 안전하게 되돌리는 방법
2. **Branching**: 독립적인 작업 공간 생성 및 관리

## 🚀 실행 방법
```bash
python main.py
```

## 📁 프로젝트 구조
* `main.py`: 프로그램 실행 엔트리 포인트
* `agent.py`: LLM 에이전트 핵심 로직 클래스
* `tools.py`: 에이전트가 사용하는 도구(날씨 등) 모음
* `workflow.py`: 작업 흐름 설정 파일
